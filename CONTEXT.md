# CONTEXT.md

## Overview

This project is a **personal portfolio website** for Sonam Wangdi Sherpa, a frontend developer with professional experience in React and Next.js.

The purpose of this website is not just to display projects, but to **position Sonam as a strong, thoughtful, and hireable frontend engineer** for recruiters, hiring managers, and technical teams in Australia.

The site must feel **premium, modern, and intentional**, reflecting both technical ability and design sensibility.

---

## Goals

### Primary Goal
Help Sonam secure:
- Frontend Developer roles
- React / Next.js roles
- Graduate / Junior–Mid Software Engineering roles
- Internship-to-full-time conversion opportunities

### Secondary Goals
- Showcase real-world and personal projects
- Communicate technical depth and design quality
- Build credibility through clarity and presentation
- Make it easy for recruiters to contact or evaluate quickly

---

## Target Audience

- Recruiters (non-technical, quick scanning)
- Engineering Managers (technical evaluation)
- Startup Founders (looking for versatile developers)
- Tech Leads (assessing code quality and thinking)

The site must work for both:
- **fast scanning (10–15 seconds)**
- **deep exploration (5–10 minutes)**

---

## Core Message

> Sonam is a frontend developer who builds clean, performant, and thoughtful user experiences using modern technologies like React and Next.js.

Supporting themes:
- Strong fundamentals
- Design awareness
- Real-world experience
- Continuous learning mindset
- Reliable and detail-oriented

---

## Design Principles

### 1. Premium Feel
- Clean layouts
- Consistent spacing
- Subtle animations (never flashy)
- High-quality typography
- Minimal but intentional color usage

### 2. Clarity Over Complexity
- No unnecessary visual noise
- Clear hierarchy
- Easy scanning
- Content-first approach

### 3. Modern Frontend Standards
- Smooth transitions
- Responsive design (mobile-first but desktop-polished)
- Dark mode support
- Fast perceived performance

### 4. Subtle Delight
- Micro-interactions
- Hover states
- Motion that enhances—not distracts

---

## Tone & Personality

- Professional but approachable
- Confident but not arrogant
- Clean and minimal
- Slightly warm (not cold corporate)

Avoid:
- Overly flashy animations
- Excessive gradients or noise
- Cluttered UI
- Buzzword-heavy language
