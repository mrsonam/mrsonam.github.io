# Sonam Wangdi Sherpa | Software Engineer Portfolio

![Portfolio Preview](./public/preview.png)

A brutalist, high-performance portfolio highlighting my work as a Software Engineer and Fullstack Developer. Designed with an "Architectural Monolith" aesthetic, the interface prioritizes bold typography, stark contrast, and smooth micro-interactions.

## 01 DESIGN SYSTEM: "ARCHITECTURAL MONOLITH"

The visual language of this portfolio breaks away from standard soft UIs and glassmorphism. It leverages a Swiss-Brutalist approach:

*   **Typography First:** Heavy reliance on mass-scale, responsive typography (`Manrope` for display, `Inter` for body) to create structure instead of traditional containers.
*   **High Contrast Palette:** 
    *   **Bone (`#F5F5F0`)**: A warm, off-white background that prevents eye strain while keeping it clinical.
    *   **Jet Black (`#0a0a0a`)**: Deep foreground text for maximum legibility.
    *   **Oxblood Red (`#8B0000`)**: A surgical accent color used sparingly for interactive elements and terminal cursors.
*   **Zero-Radius:** Absolute rigidity. All border-radius attributes are set to `0px` globally to reinforce the "Monolith" theme.
*   **Micro-Animations:** Fluid, purposeful animations using Framer Motion (including terminal-style blinking cursors) to bring the static design to life.

## 02 TECH STACK

This portfolio is engineered for speed, SEO, and maintainability.

*   **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **Animation:** [Framer Motion](https://www.framer.com/motion/)
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **Components:** [Radix UI primitives via shadcn/ui](https://ui.shadcn.com/)

## 03 LOCAL SETUP

To run this project locally, clone the repository and install the dependencies:

```bash
# Clone the repository
git clone https://github.com/mrsonam/mrsonam.github.io.git

# Navigate into the project
cd mrsonam.github.io

# Install dependencies (npm recommended)
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to witness the execution.
